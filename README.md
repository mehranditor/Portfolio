# Amirali Daneshvar - Creative Developer Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, education, and certifications as a passionate creative developer and Computer Engineering student at Biruni University, Istanbul, Turkey.

## Overview

This portfolio is a dynamic, responsive web application built with **HTML, CSS, and JavaScript**. It features a sleek, modern design with smooth animations and interactive elements, inspired by platforms like LinkedIn. The site includes sections for my bio, skills, featured projects, contact information, and dedicated pages for certificates and education.

## Features

- **Home:** A hero section with an animated gradient background and typing effect for my name.
- **About Me:** Details about my background, languages (English, Turkish, Persian), and hobbies (e.g., playing the violin).
- **Skills:** Organized into Programming Languages, Technical Skills, and Soft Skills with hover effects.
- **Featured Projects:** A carousel showcasing three projects with images, tech stacks, and links (e.g., "Email Shield" linked to LinkedIn).
- **Contact:** Social media links (LinkedIn, GitHub) with hover animations and sound effects.
- **Certificates:** A dedicated page to add and view certificates dynamically using localStorage.
- **Education:** A dedicated page to add and view education entries dynamically using localStorage.
- **Responsive Design:** Optimized for desktop and mobile views.
- **Interactive Elements:** Hamburger menu, music toggle, scroll animations, and carousel navigation.

## Technologies Used

- **HTML5:** Structure of the website.
- **CSS3:** Styling with gradients, animations, and Tailwind-inspired effects.
- **JavaScript:** Dynamic functionality, including carousels, form handling, and localStorage persistence.
- **Boxicons:** Icons for social links.
- **Audio:** Background music and sound effects on interactions.
